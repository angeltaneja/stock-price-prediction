Stock Price Prediction using Machine Learning & Deep Learning

An AI-powered Stock Price Prediction System that forecasts future stock prices using historical market data and machine learning/deep learning models.

This project demonstrates an end-to-end financial time series forecasting pipeline, including data collection, preprocessing, feature engineering, model training, evaluation, and deployment.
